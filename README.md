📌 Project Description

RateFlix is a responsive and interactive movie rating website that allows users to explore movies from different categories and rate them based on their preferences. The project focuses on providing a smooth user experience with modern UI design and dynamic features.

⚙️ Features Implemented
Movie categories (Hollywood, Bollywood, etc.)
Horizontal scrolling movie rows
Click-to-rate functionality using star popup
Ratings stored using Local Storage
Favorites feature with heart icon
Toggle button to show/hide favorite movies
Search functionality for filtering movies
Dark/Light mode toggle
Hero section with auto-sliding trending movies
Responsive design for mobile and desktop
Smooth animations and hover effects

🛠️ Technologies Used
HTML
CSS
JavaScript
Local Storage API

🎯 Conclusion

This project helped in understanding front-end development concepts such as DOM manipulation, event handling, responsive design, and local storage. It also improved UI/UX design skills by implementing real-world features like rating systems and favorites management.
